CSS Selectors

A

  • ::after

    This is a pseudo element that allows us to insert content on the page with CSS. The content won't actually be part of the DOM, but it will look like it is

  • :active

    This pseudo selector isused to change the appearance of a link on click. Even though it's only visible for a sec, it provides the user confirmation that the link was infact clicked. Click ME to see the :active pseudo selector in action!

    While we're on link states,remember that it's appropriate to address all link states in order. We can remember the order by remembering that love comes before hate (L[:link]-O-V[:visited]-E, H[:hover]-A[:active]-T-E).

  • Adjacent sibling

    This isn't a selector on its own, but a way of combining two selectors.

    For example, 'p + p'-the '+' sign is the adjacent sibling combinator and tells the CSS to select and paragraph tag directly after another paragraph tag with nothing in between, and apply the appropriate styles.

    It's that easy!

  • [attribute]

    Though tag names (p), IDs (#), and classes (.) are the most commonly used selectors in CSS, any of an element's attributes may be used as selectors. For example, 'input[type="text"]'. The [brackets] indicate an exact match, and will only select input elements of the type 'text'.

    Attribute selectors are case-sensitive, written inside brackets, and there are seven different types of matches you can find.

    Types of Matches:

    • [data-value] - attribute exists
    • [data-value="foo"] - attribute has an exact value
    • [data-value*="foo"] - attribute value contains this value somewhere in it
    • [data-value~="foo"] - attribute has this value in a space-separated list somewhere
    • [data-value^="foo"] - attribute value starts with this
    • [data-value=|"foo"] - attribute value has this in a dash-separated list somewhere
    • [data-value$="foo"] - attribute value ends with this

B

  • :blank

    The :blank pseudo-class builds on the :empty pseudo-class, in that it will select elements that contain HTML comments or nothing at all. Unlike :empty, :blank will also select elements that include whitespace

  • :before


    The ::before pseudo-selector is the same as after, except it adds content before the specified element

C

  • :checked

    This pseudo-class targets input elements of the type 'radio' and 'checkbox' that are in the selected state.

  • Child
